AitoCoder 命令行版本 / 使用脚本与压缩包快速安装说明

08/01, 2025

----------
若要使用脚本与压缩包安装，请将脚本与压缩包放在同一目录下启动。

----------
Windows 64-bit
以管理员身份启动 PowerShell 终端, 运行 `.\aitocoder_win.bat`

----------
macOS Apple Silicon
以管理员身份启动 Terminal, 运行 `./aitocoder_macos.sh`

----------
Linux (Debian 及其衍生, 包括 Ubuntu, deepin 等)
以管理员系统启动终端, 运行 `./aitocoder_linux.sh`
